$ |nlm> = \psi(r, \theta,\phi) \\ \psi(r,\theta,\phi) = \sqrt{\frac{3}{5}} \psi_{200}(r,\theta,\phi) + e^{i\alpha} \sqrt{\frac{2}{5}} \psi_{210}(r,\theta,\phi) \\ $ Using the wave functions from MacIntyre Table 8.2 (p. 264). We are considering a hydrogen atom, so $Z=1$

def psi2(x,z,alpha):

    psi200 = np.sqrt(3/5)*(1/np.sqrt(np.pi))*(1/2**(3/2))*(1-r/2)*np.exp(-r/2)
